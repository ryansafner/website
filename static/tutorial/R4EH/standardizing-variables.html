Chapter 7 Standardizing Variables

Easiest way is to use the scale() command as part of the mutate() command in dplyr.

library(gapminder)
gapminder<-gapminder %>% 
  mutate(s.life=scale(lifeExp),
         s.gdp=scale(gdpPercap))

summary(lm(s.life~s.gdp, data=gapminder))
